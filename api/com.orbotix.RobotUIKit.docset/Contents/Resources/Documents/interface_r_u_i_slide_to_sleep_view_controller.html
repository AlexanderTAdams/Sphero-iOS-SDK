Detailed Description

This view will dim the screen and animate in a "slide to sleep" slider at the top of the screen When the user moves the slider this view will automatically put Sphero to sleep If the user taps in the dim area outside the slider it will cancel the operatipon and animate the slider out

There are delegate callbacks on success and failure
