enum RKBatteryPowerState

Enumerated values for a robot's power state.

Enumerator:
RKBatteryPowerStateCharging 

The robot is charging. Note: an inductive charged robot will normally sleep in the charger, so you will not be able to issue a command to check the state.

RKBatteryPowerStateOK 

The robot's battery is charged state.

RKBatteryPowerStateLow 

The robot's battery is low and will soon need to be charged.

RKBatteryPowerStateCritical 

The robot's battery is critically low and the user needs to charge it immediatly.
